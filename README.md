# Supersharkz Charge Management System

A single-page charge recording system for Supersharkz Swim School â€” allowing admins to create, view, edit, and delete charge records.

## Getting Started

### Prerequisites

- Node.js >= 18
- npm >= 9

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Linting & Formatting

```bash
npm run lint
npm run format
```

## Tech Stack

- **React 19** + **TypeScript** (Vite)
- **Tailwind CSS v4** for styling
- **Zod** for schema validation
- **ESLint** + **Prettier** for code quality
